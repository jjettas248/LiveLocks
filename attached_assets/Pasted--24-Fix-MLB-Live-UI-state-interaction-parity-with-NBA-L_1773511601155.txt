#24 - Fix MLB Live UI state + interaction parity with NBA Live

Problem
The MLB Live UI is currently broken and does not match the intended LiveLocks UX.

Issues observed:
• Game header does not display teams (shows "@ — Box Score")
• Player rows are not clickable
• EdgeCards overlap with duplicated probability text
• Box score collapse toggle does nothing
• Signals and calculator mode render simultaneously

Goal
Make MLB Live behave exactly like NBA Live:

Default view:
Today's Games → Edge Signals → Box Score

Clicking a player row:
Box Score → Matchup Calculator Mode

Returning:
"← Back to Game" returns to Edge Signals view

liveGameOrchestrator
edgeCache
mlbEdgeCache
signal pipeline
engine math
calibration.ts
probability.ts
markets.ts
paEstimator.ts

Only modify:
client/src/pages/mlb-live.tsx
client/src/components/EdgeCard.tsx (if necessary)

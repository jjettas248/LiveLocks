#67 - NCAAB H1/H2 Market Repair (MASTER)

## SYSTEM
NCAAB Engine / Service Layer / UI Rendering

## OBJECTIVE
Fix two blocking issues preventing H1/H2 markets from rendering correctly:

1) Missing odds → Guardrail G7 invalidates markets
2) 2H projected panel renders in incorrect tabs

This is a surgical fix. Do NOT refactor unrelated systems.

---

## ROOT CAUSES

### Root Cause A — Missing odds → G7 invalidates markets

- SGO provides valid H1/H2 lines:
  - h1TotalLine
  - h1SpreadLine
  - h2TotalLine
  - h2SpreadLine

- SGO does NOT provide odds
- Odds API fails for H1/H2 (422 responses)

- buildMarket() enforces Guardrail G7:
  - odds must be valid (non-null, non-zero, within ±2000)
  - otherwise → unavailableMarket()

Result:
- Markets show "Unavailable"
- Book implied probability = null ("—%")
- Edge = null
- H1/H2 tabs appear broken

---

### Root Cause B — Live2HPanel renders outside active tab

Current behavior:
- <Live2HPanel> renders based only on game state:
  - play.bettingWindow === "HALFTIME"
  - OR play.half === 2

Missing:
- No marketTab gating

Result:
- 2H projected panel appears inside:
  - Full Game tab
  - 1st Half tab

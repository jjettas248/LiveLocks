✅ Objective

Add a local simulation mode that is:

Admin-only
Toggleable from Admin UI
Fully isolated from production users
Feeds the engine with mock inputs instead of live data

This gives you instant testing without waiting for games.

🧱 Architecture Overview
Flow
Admin UI toggle → /api/admin/simulation-config
                → server stores state (memory)
                → engine checks flag
                → injects simulated inputs
🔐 Step 1 — Server-side simulation state (admin only)
Add global state (server)
// server/simulationState.ts

export type SimulationConfig = {
  enabled: boolean;
  scenario: string;
};

let simulationConfig: SimulationConfig = {
  enabled: false,
  scenario: "neutral"
};

export function getSimulationConfig() {
  return simulationConfig;
}

export function setSimulationConfig(config: SimulationConfig) {
  simulationConfig = config;
}
Add admin-only routes
// server/routes/adminSimulation.ts

import { Router } from "express";
import { getSimulationConfig, setSimulationConfig } from "../simulationState";
import { requireAdmin } from "../auth";

const router = Router();

router.get("/simulation-config", requireAdmin, (req, res) => {
  res.json(getSimulationConfig());
});

router.post("/simulation-config", requireAdmin, (req, res) => {
  const { enabled, scenario } = req.body;

  setSimulationConfig({
    enabled: Boolean(enabled),
    scenario: scenario || "neutral"
  });

  res.json({ success: true });
});

export default router;
Register route
app.use("/api/admin", adminSimulationRouter);
🧠 Step 2 — Inject simulation into engine

Inside your NBA engine entry point:

import { getSimulationConfig } from "../simulationState";
Override inputs
const simulation = getSimulationConfig();

let gameState = liveGameState;
let playerStats = livePlayerStats;
let line = liveLine;

if (simulation.enabled) {
  const scenario = simulation.scenario;

  if (scenario === "neutral") {
    playerStats = {
      minutes: 18,
      points: 12,
      rebounds: 4,
      assists: 3,
      fgPct: 0.45,
      threePct: 0.36
    };
    line = 20.5;
  }

  if (scenario === "cold") {
    playerStats = {
      minutes: 18,
      points: 10,
      fgPct: 0.35,
      threePct: 0.25
    };
    line = 20.5;
  }

  if (scenario === "hot") {
    playerStats = {
      minutes: 18,
      points: 18,
      fgPct: 0.65,
      threePct: 0.50
    };
    line = 20.5;
  }

  gameState = {
    homeScore: 58,
    awayScore: 52,
    elapsedSeconds: 1440,
    totalGameSeconds: 2880
  };

  console.log("[SIMULATION_MODE]", scenario);
}
🎛️ Step 3 — Admin UI toggle
Add to Admin Panel
UI State
const [simulationEnabled, setSimulationEnabled] = useState(false);
const [scenario, setScenario] = useState("neutral");
Fetch current config
useEffect(() => {
  fetch("/api/admin/simulation-config")
    .then(res => res.json())
    .then(data => {
      setSimulationEnabled(data.enabled);
      setScenario(data.scenario);
    });
}, []);
Update config
const updateSimulation = async () => {
  await fetch("/api/admin/simulation-config", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      enabled: simulationEnabled,
      scenario
    })
  });
};
UI Component
<div className="p-4 border border-white/10 rounded-lg">
  <h3 className="text-sm font-semibold mb-2">Simulation Mode</h3>

  <label className="flex items-center gap-2">
    <input
      type="checkbox"
      checked={simulationEnabled}
      onChange={(e) => setSimulationEnabled(e.target.checked)}
    />
    Enable Simulation
  </label>

  <select
    value={scenario}
    onChange={(e) => setScenario(e.target.value)}
    className="mt-2 bg-black border border-white/10 p-2 rounded"
  >
    <option value="neutral">Neutral</option>
    <option value="cold">Cold Shooting</option>
    <option value="hot">Hot Shooting</option>
  </select>

  <button
    onClick={updateSimulation}
    className="mt-3 px-3 py-1 bg-green-600 rounded"
  >
    Apply
  </button>
</div>

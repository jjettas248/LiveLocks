ADDENDUM — TASK #24 ENGINE CORRECTIONS

Two corrections must be applied to the calibration upgrade to prevent false edges and probability distortion.

----------------------------------------
CORRECTION 1 — USE REAL SPORTSBOOK ODDS
----------------------------------------

The current plan assumes -110 odds when computing sportsbook implied probability.

REMOVE any fixed calculation like:

sportsbookImplied = 100 / (110 + 100)

Instead derive implied probability from the actual odds returned by the sportsbook feed.

Add helper:

function americanOddsToProb(odds: number) {

  if (odds < 0) {
    return Math.abs(odds) / (Math.abs(odds) + 100)
  }

  return 100 / (odds + 100)
}

Then compute:

sportsbookImplied = americanOddsToProb(bookOdds) * 100

edge = modelProbability - sportsbookImplied

Edge sanity rule:

noSignal = edge < 3

Attach noSignal to the response object and debug object.

----------------------------------------
CORRECTION 2 — CALIBRATION ORDER
----------------------------------------

Calibration must be the LAST probability transformation.

The correct probability pipeline is:

expected stat

→ clamp [2,98]

→ return response

Do NOT run calibration before variance filters.

CORRECTION 3 — PROBABILITY CLAMPING

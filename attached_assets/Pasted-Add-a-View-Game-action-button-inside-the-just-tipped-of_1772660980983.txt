MULTIPLE GAMES TIPPING OFF SIMULTANEOUSLY:
  - If 2+ games go live in the same refresh cycle:
      show one toast per game, stacked vertically
      each toast offset by 8px + (index * 72px) from bottom
      each dismisses independently
      max 3 toasts visible at once — queue the rest

TOAST STATE:
  const [toasts, setToasts] = useState([])
  
  const addToast = (game) => {
    const id = `toast-${game.id}-${Date.now()}`
    setToasts(prev => [...prev, { id, game }])
    setTimeout(() => dismissToast(id), 6000)
  }
  
  const dismissToast = (id) => {
    setToasts(prev => prev.filter(t => t.id !== id))
  }

Trigger addToast inside the useEffect that watches for

FILTER LOGIC

When a book is selected (not "all"):
  Only show play cards where that book has 
  a line available for that prop.

Each play in the ranked list has a lines array 
from the props API — something like:
  play.lines = [
    { book: "draftkings", line: 8.5, overOdds: -115 },
    { book: "fanduel", line: 8.5, overOdds: -110 },
    ...
  ]

  Find the exact field name for book identifier 
  in the existing play data structure.

FILTER FUNCTION:

  const filterByBook = (plays, selectedBook) => {
    if (selectedBook === "all") return plays
    
    const bookKeyMap = {
      "dk":   ["draftkings", "draft_kings", "dk"],
      "fd":   ["fanduel", "fan_duel", "fd"],
      "mgm":  ["betmgm", "mgm", "bet_mgm"],
      "br":   ["betrivers", "bet_rivers", "br", 
               "sugarhouse"],
      "espn": ["espnbet", "espn_bet", "espn"]
    }
    
    const validKeys = bookKeyMap[selectedBook] ?? 
      [selectedBook]
    
    return plays.filter(play => {
      const playLines = 
        play.lines ?? 
        play.bookLines ?? 
        play.odds ?? 
        []
      
      return playLines.some(line => {
        const bookId = (
          line.book ?? 
          line.bookmaker ?? 
          line.provider ?? 
          ""
        ).toLowerCase()
        
        return validKeys.some(k => bookId.includes(k))
      })
    })
  }

Apply filterByBook after all other filters:

  const filteredPlays = plays
    .filter(byPropType)
    .filter(byConfidence)
    .filter(byStat)
    .filter(play => filterByBook([play], selectedBook)
      .length > 0)
    .sort((a, b) => b.engineProb - a.engineProb)

    const getBookCount = (plays, bookKey) => {
      if (bookKey === "all") return plays.length
      return filterByBook(plays, bookKey).length
    }
  
  Count badge inside pill:
    "DK · 14"
    The count updates when other filters change
    (if prop filter = REB, DK pill shows REB 
     count only for DK)
  
  If count is 0:
    Pill shows at 40% opacity
    Still tappable (shows empty state)
    Count: "DK · 0"
  
  If count = all plays:
    No count shown — just "All" label
    "All" pill always shows no count

  Book filter resets to "all"

  book filter state

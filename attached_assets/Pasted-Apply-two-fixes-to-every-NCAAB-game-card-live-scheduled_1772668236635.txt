FIX 2: FULL GAME / 1H TOGGLE ON ALL NCAAB CARDS

Add the Full Game / 1H toggle to every NCAAB card 
that shows market data — live and pre-game cards.
Final cards do not need the toggle.

TOGGLE STATE:
  const [marketTab, setMarketTab] = useState("full")
  Default is always "full" — never "h1"

TOGGLE UI:
  Position: top of stat grid section, full width
  
  <div style={{
    display: "inline-flex",
    background: "#0f0f0f",
    borderRadius: "8px",
    padding: "4px",
    gap: "4px"
  }}>
    <button 
      onClick={() => setMarketTab("full")}
      style={{
        background: marketTab === "full" ? "#27272a" : "transparent",
        color: marketTab === "full" ? "#ffffff" : "#71717a",
        borderRadius: "6px",
        padding: "4px 12px",
        fontSize: "12px",
        fontWeight: 500,
        border: "none",
        cursor: "pointer"
      }}
    >
      Full Game
    </button>
    <button
      onClick={() => setMarketTab("h1")}
      style={{
        background: marketTab === "h1" ? "#27272a" : "transparent",
        color: marketTab === "h1" ? "#ffffff" : "#71717a",
        borderRadius: "6px",
        padding: "4px 12px",
        fontSize: "12px",
        fontWeight: 500,
        border: "none",
        cursor: "pointer"
      }}
    >
      1st Half
    </button>
  </div>

WHAT CHANGES WHEN TAB SWITCHES:

  marketTab === "full":
    Full Game Total row: full game total line
    Engine Over%: full game over probability
    Engine Under%: full game under probability
    Spread: full game spread
    Away Proj: full game projected final
    Home Proj: full game projected final
    Market buttons: full game lines
    Verdict rows: full game edge gap
    Book pills: full game O/U line from books

  marketTab === "h1":
    Full Game Total row → label changes to "H1 TOTAL"
    Value: H1 total line
    Engine Over%: H1 over probability
    Engine Under%: H1 under probability
    Spread: H1 spread
    Away Proj: H1 projected score
    Home Proj: H1 projected score
    Market buttons: H1 lines
    Verdict rows: H1 edge gap
    Book pills: H1 O/U line if available, else "H1 Lines TBD"

  GAUGE: never changes — always full game probability
  Add permanent "Full Game" label below gauge in #71717a text-xs
  This label does not change when tab switches

  If H1 data is unavailable for a field:
    Show "--" — do not break layout
    Show small note under toggle: 
    "1H lines unavailable" in #71717a text-xs italic

SCOPE:
  This toggle must exist on:
    Live game cards ✓
    Pre-game cards ✓
  
  This toggle must NOT exist on:
    Final cards ✗
    Scheduled row chips ✗
    Sidebar condensed rows ✗

  Each card has its own independent marketTab state.
  One card switching to H1 does not affect other cards.

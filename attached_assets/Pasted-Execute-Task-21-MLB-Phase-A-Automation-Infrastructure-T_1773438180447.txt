# PART 1 — GAME DISCOVERY SERVICE

File: `server/mlb/gameDiscoveryService.ts`

Define:

```ts
interface MLBGame {
  gameId: string
  homeTeam: string
  awayTeam: string
  startTime: string
  homePitcher?: string
  awayPitcher?: string
}
```

Implement:

```ts
async function discoverTodaysGames(): Promise<MLBGame[]>
```

Use MLB Stats API:

https://statsapi.mlb.com/api/v1/schedule?sportId=1

Filter games where:

```
abstractGameState === "Live" OR "Preview"
```

Use an 8-second timeout and full try/catch.

# PART 3 — DATA PULL SERVICE

File: `server/mlb/dataPullService.ts`

Create shared cache:

```ts
export const mlbGameCache = {
  gameState: {},
  contactData: {},
  pitcherContext: {},
  weather: {},
  bullpen: {}
}
```

Implement the following functions.

All must:

• use AbortSignal.timeout(8000)
• wrap in try/catch
• log errors but never throw

---

## syncGameState(gameId)

Endpoint:

```
https://statsapi.mlb.com/api/v1.1/game/{gamePk}/feed/live
```

Extract:

```
inning
isTopInning
outs
runnersOnBase
battingOrder
currentBatter
pitcherInGame
pitchCount
timesThroughOrder
```

Write to:

```
mlbGameCache.gameState[gameId]
```

---

## syncContactData(gameId)

Endpoint:

```
https://baseballsavant.mlb.com/gf?game_pk={gamePk}
```

Extract:

```
exitVelocity
launchAngle
hitDistance
hardHit%
barrel%
```

Store:

```
mlbGameCache.contactData[gameId]
```

---

## syncPitcherContext(gameId)

Use live feed.

Extract:

```
pitch mix
velocity
pitch count
timesThroughOrder
```

Detect velocity drop.

Store:

```
mlbGameCache.pitcherContext[gameId]
```

---

## syncWeather(gameId)


Store:

```
mlbGameCache.weather[gameId]
```

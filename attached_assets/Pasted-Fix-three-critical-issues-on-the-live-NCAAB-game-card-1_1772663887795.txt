3. AUTO-REFRESH SCORE EVERY 3 MINUTES

Live game scores update automatically every 3 minutes.
Manual refresh button also available.

AUTO-REFRESH LOGIC:
  useEffect(() => {
    if (game.status !== "live") return
    
    const interval = setInterval(() => {
      fetchGameData(game.id)
    }, 180000) // 3 minutes in ms
    
    return () => clearInterval(interval)
  }, [game.id, game.status])

  fetchGameData should:
    1. Hit ESPN live scoreboard endpoint for this game
    2. Update score, clock, quarter
    3. Update engine probabilities
    4. Update projected finals
    5. Trigger verdict row recalculation + flash animation
    6. NOT reset selectedMarket or marketTab state
    7. NOT collapse expanded H2H section if user opened it

VISUAL REFRESH INDICATOR:
  Small indicator in card header, right of live badge:
    Shows "Auto-refresh in [X]m [SS]s" countdown
    Format:
      > 60s: "Auto-refresh in [X]m [SS]s" zinc-500 text-xs
      < 60s: "Refreshing soon..." amber text-xs
      During fetch: "Updating..." teal text-xs 
                    with spinning loader icon (lucide Loader2)
      After fetch: brief "Updated ✓" teal text-xs 
                   fades out after 2 seconds

  Countdown resets to 3:00 after each successful fetch.
  
  const [refreshCountdown, setRefreshCountdown] = useState(180)
  
  useEffect(() => {
    if (game.status !== "live") return
    const tick = setInterval(() => {
      setRefreshCountdown(prev => {
        if (prev <= 1) {
          fetchGameData(game.id)
          return 180
        }
        return prev - 1
      })
    }, 1000)
    return () => clearInterval(tick)
  }, [game.id, game.status])

MANUAL REFRESH:

      fetchGameData for ALL live games simultaneously
      Reset all live card countdown timers to 180

SCORE ANIMATION ON UPDATE:
  When score changes after a refresh:

  If score did not change: no animation, silent update.

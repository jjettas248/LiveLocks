Implement the frontend-only paywall optimization for free users in the NBA dashboard, keeping all backend logic unchanged.

Goal:

6. Add a summary line above each game’s edge grid showing:
   - total edges detected for that game
   - how many are visible
   - how many are locked
7. Add an “Unlock All Edges” CTA below the locked section that opens the upgrade modal.
8. Do not change:
   - backend routes
   - middleware
   - DB schema
   - `FREE_PLAY_LIMIT`
   - calculator result cards
   - NCAAB tab behavior
   - MLB tab behavior

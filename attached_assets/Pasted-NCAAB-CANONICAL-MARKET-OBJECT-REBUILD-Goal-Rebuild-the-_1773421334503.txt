NCAAB CANONICAL MARKET OBJECT REBUILD

Goal
Rebuild the NCAAB engine/UI contract around a single canonical market object model.

The engine computes markets once.
The UI renders them directly.
No legacy fields.
No UI inference.
No silent fallbacks.

------------------------------------------------

MARKET CONTRACT

engineOutput.markets = {
  full_total,
  full_spread,
  h1_total,
  h1_spread,
  h2_total,
  h2_spread
}

Each key is always present.

If a sportsbook line is unavailable or the game state does not permit the market:

available = false

------------------------------------------------

MARKET OBJECT

NCAABMarket fields:

available
marketKey
label
sportsbook
bookLine
projection
modelProb
bookImpliedProb
edge
side

Rules

projection must never be rendered as a sportsbook line
modelProb must be clamped to [1, 99]
edge = modelProb - bookImpliedProb
UI must never compute edge

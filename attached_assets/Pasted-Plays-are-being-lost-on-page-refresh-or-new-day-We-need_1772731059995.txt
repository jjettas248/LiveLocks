IF Supabase:

  Run this migration or create table via Supabase

  CREATE TABLE IF NOT EXISTS plays (
    id TEXT PRIMARY KEY,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    game_id TEXT NOT NULL,
    player_id TEXT,
    player_name TEXT NOT NULL,
    team TEXT,
    sport TEXT NOT NULL DEFAULT 'nba',
    market TEXT NOT NULL,
    direction TEXT NOT NULL,

    engine_prob DECIMAL,
    book_implied DECIMAL,
    edge_gap DECIMAL,

    timestamp TIMESTAMPTZ NOT NULL,
    result TEXT DEFAULT NULL,
    final_stat DECIMAL DEFAULT NULL,
    settled_at TIMESTAMPTZ DEFAULT NULL,
    notification_sent BOOLEAN DEFAULT FALSE,
    is_duplicate_guard TEXT UNIQUE
    -- is_duplicate_guard = getPlayKey() value
    -- UNIQUE constraint prevents DB-level duplicates
  );

  CREATE INDEX idx_plays_game_date 
    ON plays(game_date DESC);
  CREATE INDEX idx_plays_result 
    ON plays(result);
  CREATE INDEX idx_plays_sport 
    ON plays(sport);

STEP 3 — API ROUTES FOR PLAYS

Create these routes using the existing API pattern:

POST /api/plays/record
  Body: play object
  Action:
    - Generate id if missing: 
        `play-${Date.now()}-${Math.random()
          .toString(36).slice(2,7)}`
    - Generate duplicateGuard from getPlayKey()
    - Upsert (insert or ignore if duplicate):
        Supabase: 
          .upsert(play, { 
            onConflict: "duplicateGuard",
            ignoreDuplicates: true 
          })
        Prisma:
          .upsert({
            where: { duplicateGuard: key },
            update: {},  // never overwrite existing
            create: play
          })
  Returns: { success: true, id, isDuplicate: bool }

GET /api/plays?sport=nba&limit=100&settled=all
  Query params:
    sport: "nba" | "ncaab" | "all"
    limit: number (default 100)
    settled: "all" | "pending" | "settled"
    date: "YYYY-MM-DD" (optional, filter by game_date)
  
  Returns: { plays: Play[], total: number }

PATCH /api/plays/:id/settle
  Body: { result, finalStat, settledAt }
  Action: Update play result fields
  Returns: { success: true, play: updatedPlay }

GET /api/plays/stats
  Returns bucket stats for model performance:
  {
    buckets: {
      "60-69": { total, hits, misses, winRate },
      "70-79": { total, hits, misses, winRate },
      "80-89": { total, hits, misses, winRate },
      "90+":   { total, hits, misses, winRate }
    },
    totalSettled: number,
    totalPending: number,
    allTimeRecord: { hits, misses, pushes }
  }

STEP 4 — WIRE PLAY RECORDING

Find every place a play is currently captured
(notification trigger, 2H plays loader, etc).

Replace in-memory writes with API call:

  // Before (wrong — in memory):
  setPlays(prev => [...prev, newPlay])

  // After (correct — persisted):
  const recordPlay = async (play) => {
    const key = getPlayKey(play)
    
    // Check in-memory dedup first (fast)
    if (recordedKeys.current.has(key)) return
    recordedKeys.current.add(key)
    
    // Persist to DB
    try {
      const res = await fetch("/api/plays/record", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          ...play,
          duplicateGuard: key
        })
      })
      const data = await res.json()
      if (data.isDuplicate) {
        console.log("[PLAYS] Duplicate blocked:", key)
      }
    } catch (err) {
      console.error("[PLAYS] Record failed:", err)
      // Do not throw — notification should still fire
    }
  }

STEP 5 — LOAD PLAYS FROM DB ON MOUNT

Replace useState initialization with DB fetch:

  const [plays, setPlays] = useState([])
  const [playsLoading, setPlaysLoading] = useState(true)
  
  useEffect(() => {
    const loadPlays = async () => {
      try {
        const res = await fetch(
          "/api/plays?sport=nba&limit=100"
        )
        const data = await res.json()
        setPlays(data.plays ?? [])
      } catch (err) {
        console.error("[PLAYS] Load failed:", err)
      } finally {
        setPlaysLoading(false)
      }
    }
    loadPlays()
  }, [])

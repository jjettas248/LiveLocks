STEP 2 — SINGLE SOURCE OF TRUTH FOR PLAY RECORDING

There must be exactly ONE place in the code that
calls recordPlay(). All others must be removed.

The correct trigger:
  Record a play ONCE when it is first detected
  as a high-confidence signal (>= 80%).
  
  NOT on every refresh.
  NOT in a render function.
  NOT in multiple places.

CORRECT PATTERN:

  // Module-level set — persists across renders
  // Resets only on page refresh (intentional)
  const recordedPlayKeys = new Set()
  
  const recordPlayOnce = async (play) => {
    const key = getPlayKey(play)
    
    // Guard 1: in-memory check (fastest)
    if (recordedPlayKeys.has(key)) {
      return  // already recorded this session
    }
    
    // Guard 2: DB unique constraint will also
    // prevent duplicates (belt + suspenders)
    recordedPlayKeys.add(key)
    
    // Now record
    await recordPlay(play)
  }

  // getPlayKey must produce consistent output
  const getPlayKey = (play) => {
    const date = new Date(play.timestamp)
      .toISOString()
      .split("T")[0]  // "2026-03-05"
    
    return [
      String(play.playerId ?? "").trim(),
      String(play.market ?? "").toUpperCase().trim(),
      String(play.line ?? ""),
      String(play.direction ?? "").toLowerCase().trim(),
      String(play.gameId ?? "").trim(),
      date
    ].join("|")
  }

MOVE recordedPlayKeys OUTSIDE COMPONENT:
  If currently inside a React component:
    const recordedPlayKeys = new Set()  ← WRONG
    // Resets on every render
  
  Move to module scope:
    // At top of file, outside any function:
    const recordedPlayKeys = new Set()  ← CORRECT
    // Persists for lifetime of page session

STEP 5 — CLEAN EXISTING DUPLICATES IN DB

One-time cleanup of existing duplicate plays.
Run this via the Settle Now button or a separate
admin "Clean Duplicates" button.

  const cleanDuplicatePlays = async () => {
    // Get all plays
    const allPlays = await fetchAllPlays()
    
    // Group by duplicateGuard key
    const seen = new Map()
    const toDelete = []
    
    allPlays.forEach(play => {
      const key = getPlayKey(play)
      
      if (seen.has(key)) {
        // Keep the one with highest prob
        // or earliest timestamp
        const existing = seen.get(key)
        if (play.prob > existing.prob) {
          toDelete.push(existing.id)
          seen.set(key, play)
        } else {
          toDelete.push(play.id)
        }
      } else {
        seen.set(key, play)
      }
    })
    
    // Delete duplicates
    if (toDelete.length > 0) {
      console.log(
        "[CLEAN] Deleting", toDelete.length, 
        "duplicate plays"
      )
      await deletePlays(toDelete)
    }
    
    return {
      removed: toDelete.length,
      remaining: allPlays.length - toDelete.length
    }
  }

MLB PROBABILITY MODEL STRUCTURE

The engine must calculate probability using plate appearance modeling.

Do NOT use simple averages.

Use this structure:

--------------------------------------------------

STEP 1 — PROJECT REMAINING PLATE APPEARANCES

Expected remaining PA:

remainingPA =
(9 - currentInning) * 0.44

Adjust for batting order:

slot 1–2 → +0.4 PA
slot 3–5 → +0.2 PA
slot 6–9 → −0.1 PA

Add current game pace adjustment:

paceFactor =
(currentRuns / leagueAvgRuns)

remainingPA *= paceFactor

Clamp:

remainingPA between 1.0 and 3.5

--------------------------------------------------

STEP 2 — PLAYER HIT PROBABILITY

Use player batting stats:

hitProbability =
playerHits / playerAB

Blend with league average:

leagueHitRate = .243

blendedHitRate =
(0.65 * hitProbability) +
(0.35 * leagueHitRate)

--------------------------------------------------

STEP 3 — PROBABILITY OF AT LEAST ONE HIT

Using binomial distribution:

P(hit ≥ 1) =
1 - (1 - blendedHitRate) ^ remainingPA

--------------------------------------------------

STEP 4 — EXPECTED HITS

expectedHits =
remainingPA * blendedHitRate

--------------------------------------------------

STEP 5 — EDGE CALCULATION

Book implied probability:

impliedProbability =
americanOddsToProbability(bookOdds)

Edge:

edge =
modelProbability - impliedProbability

Refactor the MLB Live page to support a dual-mode experience identical to NBA Live while preserving the existing signals system.

The MLB page must support two modes:

1) automatic edge signals
2) manual player probability calculator

Do NOT modify the orchestrator, edgeCache, triggerEngine, or signal pipeline.

--------------------------------------------------

ENGINE BOUNDARY RULE

All probability calculations must remain server-side.

The frontend must NOT calculate remaining plate appearances or probabilities.

The frontend should only send:

playerId
selected sportsbook line
current inning
current stats
batting order slot

The backend engine will calculate remainingPA, probabilities, and edge.

--------------------------------------------------

FEATURE 1 — OPEN CALCULATOR

Each player row in the box score must be clickable.

onClick={() => setSelectedPlayer(player)}

Selecting a player switches the UI into calculator mode.

Add state:

const [selectedPlayer, setSelectedPlayer] = useState(null)

Add button:

← Back to Game

which clears selectedPlayer.

const [boxExpanded, setBoxExpanded] = useState(true)

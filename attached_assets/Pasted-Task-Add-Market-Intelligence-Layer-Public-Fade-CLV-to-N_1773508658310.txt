Task — Add Market Intelligence Layer (Public Fade + CLV) to NCAAB Engine

Goal
Upgrade the NCAAB betting engine to incorporate market intelligence signals derived from sportsbook line movement and public betting percentages.

The engine should detect:

• Public-driven line inflation
• Closing Line Value (CLV)
• Live price improvements relative to closing market consensus

These signals must augment the existing model-based edge calculation without replacing it.

No UI changes are required in this task.

----------------------------------------

Extend the NCAABMarket interface with optional market intelligence fields.

Add:

clvEdge?: number

fadeRecommended?: boolean
adjustedEdgeScore?: number

These fields must remain optional so existing engine consumers are not affected.

SECTION 3 — Line Movement Detection Helper

Add helper function inside ncaabEngine.ts.

function lineMovedTowardPublic(openLine: number, closingLine: number, publicPct: number): boolean {

  if (publicPct >= 65) {
    return Math.abs(closingLine) > Math.abs(openLine)
  }

  return false
}

This detects cases where public betting pressure pushed the line.

----------------------------------------

SECTION 4 — Closing Line Value Calculation

Add helper function:

function computeCLV(closingLine: number | null, liveLine: number | null): number | null {

  if (
    closingLine === null ||
    liveLine === null ||
    !isFinite(closingLine) ||
    !isFinite(liveLine)
  ) {
    return null
  }

  return Math.abs(closingLine) - Math.abs(liveLine)
}

Positive CLV means the bettor is receiving a better price than the closing market.

SECTION 5 — Detect Public Inflation

Inside runNCAABEngine after reading engineInput:

const publicInflationSpread =
  engineInput.publicSpreadPct !== undefined &&
  engineInput.openSpread !== undefined &&
  engineInput.closingSpread !== undefined &&
  lineMovedTowardPublic(
    engineInput.openSpread,
    engineInput.closingSpread,
    engineInput.publicSpreadPct
  )

const publicInflationTotal =
  engineInput.publicTotalPct !== undefined &&
  engineInput.openTotal !== undefined &&
  engineInput.closingTotal !== undefined &&
  lineMovedTowardPublic(
    engineInput.openTotal,
    engineInput.closingTotal,
    engineInput.publicTotalPct
  )

SECTION 8 — Edge Score Enhancement

let marketEdgeBoost = 0

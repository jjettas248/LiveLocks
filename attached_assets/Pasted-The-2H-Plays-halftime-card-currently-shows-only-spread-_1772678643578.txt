3. VERDICT ROWS (auto-generated)

Same logic as live card:

  const edgeGap = Math.abs(engineProb - bookImplied)
  const edgeSide = engineProb > bookImplied 
    ? "Over" : "Under"
  const edgeLabel =
    edgeGap >= 20 ? `Strong ${edgeSide} EV` :
    edgeGap >= 10 ? `Lean ${edgeSide} EV` :
    "Neutral — No Edge"

Row 1 — EV:
  Left: edgeLabel colored teal (Over) or red (Under)
  Sub: "Engine [X]% vs Book [Y]%"
  Right: amber pill "+[X.X]pp Edge"
  Left border: 3px teal or red

Row 2 — CLV:
  Left: "[Side] CLV"
  Sub: "Closing line value signal"
  Right: directional badge teal/red or "Even" zinc

Both rows below threshold (<5pp):
  40% opacity + "Edge below threshold — monitoring"

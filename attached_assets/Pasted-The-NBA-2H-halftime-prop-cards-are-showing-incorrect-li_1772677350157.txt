STEP 2 — IDENTIFY THE BUG

Based on the screenshot, the visible issue is:

  Card shows: "Assists O2  Live Line  H1: 5"
  
  "O2" = the prop line (Over 2 Assists)
  "H1: 5" = current H1 stat (player has 5 assists 
             at halftime — this is NOT the line)

The line display and the current stat display 
appear to be confused or swapped.

Confirm which field is which in the data object:
  - Find the field storing the prop line value
    (the number the bet is placed on — e.g. 2.0)
  - Find the field storing the current H1 stat
    (how many the player has so far — e.g. 5)
  - These must NOT be mixed up in the display

Correct display format:
  "[Market] [Direction][Line]  Live Line  H1: [current stat]"
  Example: "Assists O2  Live Line  H1: 5"
  
  This IS correct formatting — O2 is the line.
  
  What IS broken is whether the line value "2" is 
  coming from the actual live halftime prop line 
  from the book, or from stale pre-game data.

The NCAAB game list must display as a horizontal scrolling 
strip identical to the NBA Live "Today's Games" strip.

---

STEP 1 — FIND THE NBA STRIP COMPONENT

Locate the existing Today's Games horizontal strip 
used in the NBA Live tab.

Find:
  - The component file name
  - The props it accepts
  - How games data is passed to it
  - How individual chips are rendered
  - How chip clicks are currently handled

Do NOT rebuild this component from scratch.
Reuse it entirely.

---

STEP 2 — MAKE THE STRIP ACCEPT A GAMES PROP

If the NBA strip component is hardcoded with NBA games:
  Refactor it to accept games and sport as props:
  
  <TodaysGamesStrip 
    games={games} 
    sport="ncaab"
    onChipClick={handleChipClick}
  />

  The component internally renders the same chip layout 
  regardless of sport — only the data source changes.

If it already accepts a games prop:
  Skip refactor, go straight to Step 3.

Do NOT duplicate the component.
One component, two usages.

---

STEP 3 — WIRE NCAAB GAMES INTO THE STRIP

In the NCAAB Live tab:
  Import the same TodaysGamesStrip component
  Pass NCAAB games array to it
  Pass handleChipClick as the click handler

  <TodaysGamesStrip
    games={ncaabGames}
    sport="ncaab"
    onChipClick={handleChipClick}
  />

  Where ncaabGames is the same games array already 
  used to render the vertical list below.

---

STEP 4 — CHIP DISPLAY FOR NCAAB

Each chip in the NCAAB strip shows:

  Layout (match NBA chip exactly):
    Left:   AWAY team abbreviation, font-bold white
    Center: score with colored separator
            "0 – 0" if not started (blue, match NBA style)
            Live score in white font-bold
            Final score in zinc-400
    Right:  HOME team abbreviation, font-bold white
    Below:  tipoff time (scheduled) or clock (live)

  Live chip:
    Score: white font-bold
    Clock below: green text text-xs "H1 14:37"
    Green pulsing dot left of clock
    Chip border: zinc-700 with rgba(0,212,170,0.2) glow

  Scheduled chip:
    Score: "0–0" in #3b82f6 blue (match NBA style)
    Time below: #71717a text-xs

  Final chip:
    Score: zinc-400 muted
    "Final" below: zinc-500 text-xs

  Selected chip (expandedGameId matches):
    Border: #00d4aa 1.5px teal
    Bg: #1f1f1f
    Transition: 150ms ease

---

STEP 5 — CHIP CLICK BEHAVIOR

onChipClick handler:
  1. setExpandedGameId(game.id)
  2. Wait 50ms for expand animation to start
  3. Scroll to game row:
       rowRefs.current[game.id]?.scrollIntoView({
         behavior: "smooth",
         block: "start"
       })
  
  Each game row in the list below must have:
    ref={el => rowRefs.current[game.id] = el}
    className includes "scroll-mt-24"

---

STEP 6 — REMOVE OLD VERTICAL LIST

Once the horizontal strip is confirmed rendering 
NCAAB games correctly:

  Remove the old vertical grouped-by-time list entirely.
  Do not keep both rendering simultaneously.
  Do not comment it out — delete it.

  The new layout for NCAAB Live tab from top to bottom:
    1. Horizontal strip (TodaysGamesStrip)
    2. Grouped game rows with inline expandable cards
    3. Sidebar (when a game is expanded)

  The grouped-by-time section headers (5:00 PM CT, etc.)
  remain below the strip as section dividers for the 
  expandable card list — only the OLD vertical chip/row 
  list inside those groups is removed and replaced by 
  the strip above.

---

STEP 7 — SECTION HEADERS BELOW STRIP

Keep time-grouped section headers for the expandable 
card list below the strip. These are NOT removed.

Each section header:
  "[TIME] CT" small caps zinc-500
  Full width zinc-800 divider line
  
Games within each section:
  Single full-width row (not the old multi-column grid)
  bg #111111, border zinc-800, rounded-lg
  cursor pointer, hover border zinc-600
  onClick={() => handleRowClick(game.id)}
  ref={el => rowRefs.current[game.id] = el}
  className="scroll-mt-24"

  Left: "[AWAY full name] @ [HOME full name]" #ffffff bold
        Score or time below in #a1a1aa text-sm
  Right: status badge (Live green / Scheduled zinc / Final muted)

  Clicking row = same expand behavior as chip click
  Both chip and row expand the same card — 
  they share expandedGameId state.

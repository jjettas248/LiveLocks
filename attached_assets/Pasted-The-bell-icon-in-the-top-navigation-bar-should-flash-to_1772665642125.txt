The bell icon in the top navigation bar should flash 
to prompt users to add their phone number for alerts.
Stop flashing once they opt in or explicitly opt out.

SMS OPT-IN STATE:
  Track in a top-level state (or localStorage for persistence):
  
  type SmsStatus = "unprompted" | "opted-in" | "opted-out"
  
  const [smsStatus, setSmsStatus] = useState<SmsStatus>(() => {
    return (localStorage.getItem("smsStatus") as SmsStatus) 
      || "unprompted"
  })
  
  Persist every status change to localStorage:
    localStorage.setItem("smsStatus", newStatus)

BELL FLASH BEHAVIOR:

  smsStatus === "unprompted":
    Bell flashes on every tab switch
    Flash animation:
      @keyframes bellFlash {
        0%   { color: #71717a; filter: drop-shadow(0 0 0px transparent) }
        40%  { color: #f59e0b; filter: drop-shadow(0 0 6px #f59e0b) }
        100% { color: #71717a; filter: drop-shadow(0 0 0px transparent) }
      }
      animation: bellFlash 800ms ease 2
      (fires twice on each tab switch, then stops until next switch)
    
    Bell also has a persistent small amber dot indicator:
      8px dot, bg #f59e0b, rounded-full
      Position: top-right corner of bell icon
      animate-ping, slow (2s cycle)
      Always visible while smsStatus === "unprompted"

  smsStatus === "opted-in":
    No flash, no amber dot
    Bell icon color: zinc-400 static
    Bell icon shows a small green dot instead:
      6px, bg green-400, static (no ping)
      Indicates active SMS alerts on

  smsStatus === "opted-out":
    No flash, no dot of any kind
    Bell icon color: zinc-600 static (slightly dimmed)
    No animation ever

TAB SWITCH TRIGGER:
  Every time activeTab changes:
    if (smsStatus === "unprompted") {
      triggerBellFlash()
    }
  
  triggerBellFlash:
    Removes and re-adds the flash CSS class to force 
    animation replay:
      bellRef.current?.classList.remove("bell-flash")
      void bellRef.current?.offsetWidth // force reflow
      bellRef.current?.classList.add("bell-flash")

PHONE VALIDATION:
  const isValidPhone = (phone: string) => {
    const cleaned = phone.replace(/\D/g, "")
    return cleaned.length === 10 || 
           (cleaned.length === 11 && cleaned[0] === "1")
  }

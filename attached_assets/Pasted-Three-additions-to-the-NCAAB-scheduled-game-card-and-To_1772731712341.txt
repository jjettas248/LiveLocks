1. OPENING LINE ON GAME CHIP IN TODAY'S GAMES STRIP

Add O/U line display to scheduled game chips
in the horizontal strip.

FIND the TodaysGamesStrip chip component.
Chips currently show: team abbrs, score or time.

SCHEDULED CHIP UPDATE:

  Current scheduled chip layout:
    [AWAY abbr]  [time]  [HOME abbr]
    
  New scheduled chip layout:
    [AWAY abbr]  [time]  [HOME abbr]
    [O/U line centered below time]

  O/U line display:
    Source: game.odds?.overUnder (already fetched 
    when card expanded — but chips need it too)
    
    Fetch strategy for chips:
      Do NOT fetch odds for every chip on load.
      That is 40+ API calls for a full slate.
      
      Instead use a lazy batch approach:
        When chips first render, mark odds as unfetched.
        When user views the strip (on tab focus or mount),
        fetch odds for the first 8 visible chips only.
        Fetch remaining chips as user scrolls strip.
      
      Use IntersectionObserver on each chip:
        When chip enters viewport for first time:
          if (!game.odds && !game.oddsFetching) {
            fetchChipOdds(game.id)
          }
      
      fetchChipOdds: lightweight fetch of just overUnder
        Uses same fetchGameOdds already built.
        After fetch: update game.odds in state.

  Chip O/U display states:
    Loading: "--" zinc-600 text-xs (placeholder)
    Loaded: "O/U [line]" zinc-400 text-xs font-mono
      e.g. "O/U 138.5"
    Unavailable: hide entirely (no empty space)

  Chip layout with line (scheduled only):
    Container: min-width 160px (NCAAB), flex-col
    
    Top row: [AWAY] [score/time] [HOME]
      Existing layout unchanged
    
    Bottom row (new, scheduled only):
      Centered: "O/U [line]"
      zinc-400 text-xs font-mono
      Appears only when odds.overUnder exists
      Does NOT appear on live or final chips
    
    Chip height: add 4px when line shows
      min-height goes from 52px to 60px
      Transition: height 150ms ease

  Live chip: no O/U line shown
    Live chips show score + clock — no room
    
  Final chip: no O/U line shown
    Final chips show final score only

  confidence and actual value — important signal.

3. SHARP MONEY INDICATOR

Show when ESPN win percentage significantly 
diverges from what the spread mathematically implies.
This is a classic sharp action signal.

SHARP MONEY CALCULATION:

  const detectSharpMoney = (odds) => {
    if (!odds) return null
    
    const espnHomeWinPct = 
      odds.homeTeamOdds?.winPercentage
    const spreadDetails = odds.spreadDetails ?? ""
    
    if (!espnHomeWinPct) return null
    
    // Parse spread value
    const spreadMatch = 
      spreadDetails.match(/([+-]?\d+\.?\d*)$/)
    if (!spreadMatch) return null
    
    const spreadValue = parseFloat(spreadMatch[1])
    
    // Convert spread to implied win probability
    // Rule of thumb: each point = ~3% win probability
    // Home team at pick'em = ~53% (home advantage)
    const spreadImpliedWinPct = spreadValue < 0
      ? 53 + (Math.abs(spreadValue) * 3)  // home favored
      : 53 - (Math.abs(spreadValue) * 3)  // away favored
    
    // Clamp to 5-95 range
    const clampedSpreadImplied = Math.min(
      Math.max(spreadImpliedWinPct, 5), 95
    )
    
    // Gap between ESPN predictor and spread implied
    const gap = espnHomeWinPct - clampedSpreadImplied
    
    // Significant divergence threshold: 8+ points
    if (Math.abs(gap) < 8) return null
    
    const sharpSide = gap > 0 
      ? "home"   // ESPN says home much better than spread
      : "away"   // ESPN says away much better than spread
    
    const strength = Math.abs(gap)
    
    return {
      detected: true,
      sharpSide,
      gap: parseFloat(gap.toFixed(1)),
      strength,
      label: strength >= 15 
        ? "Strong Sharp Signal"
        : "Sharp Money Signal",
      // Which team name to display
      teamName: sharpSide === "home"
        ? odds.homeTeamName
        : odds.awayTeamName
    }
  }

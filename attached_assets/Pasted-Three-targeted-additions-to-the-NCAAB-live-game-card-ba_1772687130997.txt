TEAM TOTAL VERDICT LOGIC:

  const getTeamTotalVerdict = (
    teamProj, line, direction, isEstimated, game
  ) => {
    const engineProb = calculateTeamTotalProb(
      teamProj, line, game
    )
    
    // If line is estimated, reduce confidence
    // Estimated lines have higher uncertainty
    const adjustedProb = isEstimated
      ? 50 + ((engineProb - 50) * 0.6)
      // Compress toward 50% by 40%
      : engineProb
    
    const bookImplied = isEstimated
      ? 50  // no real book to compare against
      : calculateBookImplied(
          direction === "over" ? -110 : -110
          // default juice if no team total odds
        )
    
    const edgeGap = Math.abs(adjustedProb - bookImplied)
    const edgeSide = adjustedProb > bookImplied 
      ? "Over" : "Under"
    
    return {
      engineProb: adjustedProb,
      bookImplied,
      edgeGap,
      edgeSide,
      edgeLabel: edgeGap >= 20 
        ? `Strong ${edgeSide} EV`
        : edgeGap >= 10 
        ? `Lean ${edgeSide} EV`
        : "Neutral — No Edge",
      isEstimated
    }
  }

2. H2H HISTORY — CURRENT SEASON FIRST, 
   LAST SEASON FALLBACK

The H2H fetch must try current season first,
only extend to last season if not enough games.

MINIMUM THRESHOLD: 2 H2H games needed.
If current season has 2+ games → use current only.
If current season has 0-1 games → extend to last season.
Always show maximum 3 H2H games total.

CURRENT SEASON ESPN SCHEDULE URL:
  Appending season year to schedule endpoint:
  
  const currentYear = new Date().getFullYear()
  // NCAA season: 2024-25 season = year 2025
  const seasonYear = new Date().getMonth() >= 7
    ? currentYear + 1  // Aug-Dec: upcoming season year
    : currentYear      // Jan-Jul: current season year

  const currentSeasonUrl =
    `https://site.api.espn.com/apis/site/v2/sports/` +
    `basketball/mens-college-basketball/teams/` +
    `${homeTeamId}/schedule?season=${seasonYear}`

  const lastSeasonUrl =
    `https://site.api.espn.com/apis/site/v2/sports/` +
    `basketball/mens-college-basketball/teams/` +
    `${homeTeamId}/schedule?season=${seasonYear - 1}`

FETCH FLOW:

  const fetchH2HHistory = async (
    homeTeamId, awayTeamId
  ) => {
    
    // Step 1: Try current season
    const currentSeason = await fetchSchedule(
      homeTeamId, currentSeasonUrl
    )
    const currentH2H = filterH2H(
      currentSeason, awayTeamId
    )
    
    // Step 2: Check if enough data
    if (currentH2H.length >= 2) {
      return currentH2H.slice(0, 3)
    }
    
    // Step 3: Not enough — extend to last season
    const lastSeason = await fetchSchedule(
      homeTeamId, lastSeasonUrl
    )
    const lastH2H = filterH2H(lastSeason, awayTeamId)
    
    // Step 4: Combine, deduplicate, sort by date desc
    const combined = [...currentH2H, ...lastH2H]
      .filter((game, index, self) =>
        index === self.findIndex(g => g.id === game.id)
      )
      .sort((a, b) => 
        new Date(b.date) - new Date(a.date)
      )
      .slice(0, 3)
    
    return combined
  }

  const filterH2H = (scheduleData, awayTeamId) => {
    return scheduleData.events
      ?.filter(event => {
        const opponents = event.competitions?.[0]
          ?.competitors ?? []
        return opponents.some(c => 
          c.team?.id === awayTeamId &&
          event.competitions?.[0]?.status
            ?.type?.completed === true
        )
      }) ?? []
  }

3. BOOK LINE PRIORITY + "Est." LABEL

Team total lines must use this priority order:

PRIORITY 1 — Book line from existing API response:

  Scan game summary for team total lines.
  ESPN sometimes returns team totals in:
    summary?.teamTotals
    summary?.pickcenter?.[0]?.teamTotals
    comp?.odds?.[0]?.teamTotals
  
  Check all three locations.
  If any returns a value for homeTeam or awayTeam:
    Use it directly
    isEstimated = false
    No "Est." label shown

PRIORITY 2 — Derived from projection (fallback):

  Only use if Priority 1 returns nothing.
  
  const deriveTeamTotalLine = (proj) => {
    if (!proj || proj <= 0) return null
    // Round to nearest 0.5
    return Math.round(parseFloat(proj) * 2) / 2
  }
  
  isEstimated = true
  Show "Est." label in team total header
